# Политика конфиденциальности

## Telegram Photo Cloud

Последнее обновление: Декабрь 2024

### Сбор данных

Приложение Telegram Photo Cloud:
- НЕ собирает персональные данные
- НЕ передаёт данные третьим лицам
- НЕ использует аналитику

### Хранение данных

Все данные хранятся локально на вашем устройстве:
- Настройки приложения
- История загрузок
- Токен бота (используется только для отправки фото)

### Разрешения

Приложение запрашивает:
- **Доступ к фото** — для чтения фотографий для загрузки
- **Интернет** — для отправки фото в Telegram
- **Уведомления** — для показа прогресса загрузки

### Telegram

Фотографии отправляются напрямую в ваш личный канал Telegram через Telegram Bot API. 
Мы не имеем доступа к вашим фотографиям или каналу.
